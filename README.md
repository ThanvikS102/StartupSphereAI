# StartupSphere AI

A full-stack startup intelligence platform built using:

## Tech Stack

- Next.js
- TypeScript
- Spring Boot
- PostgreSQL
- Maven

## Features

- Startup Directory
- Search Functionality
- Industry Filters
- Country Filters
- Company Detail Pages
- Analytics Dashboard (Coming Soon)

## Setup

### Backend

cd backend
mvn spring-boot:run

### Frontend

cd frontend
npm install
npm run dev
